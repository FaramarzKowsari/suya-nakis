## About the Project

**Suya Nakış** is an interactive digital Ebru studio that explores the meeting point of traditional Turkish marbling, creative coding, procedural graphics, digital storytelling and cultural technology.

The project allows users to place coloured drops on a digital water surface, reshape them with stylus and comb tools, adjust brush precision, observe an automated narrated demonstration and export the finished composition as a PNG image.

Rather than reproducing Ebru as a static digital image, the project transforms the creative process into an interactive browser-based experience.

## Cultural Context

Ebru is the traditional Turkish art of creating colourful patterns by sprinkling and brushing pigments on prepared water and transferring the resulting composition to paper.

The art of Ebru is included on UNESCO's Representative List of the Intangible Cultural Heritage of Humanity.

Official UNESCO reference:

https://ich.unesco.org/en/RL/ebru-turkish-art-of-marbling-00644

This project is an independent cultural-technology work. It is not affiliated with or endorsed by UNESCO.

## Key Features

### Interactive Paint Drops

Each digital paint drop is represented by a 300-vertex contour, allowing it to react to newly placed colours and interactive tools.

### Drop Tool

Place coloured circular forms on the digital water surface. Each new drop displaces existing shapes and creates layered marbling effects.

### Stylus Tool

Drag through the composition to stretch and reshape the coloured forms.

### Comb Tool

Apply multiple parallel interactions to create repeated, comb-like marbling patterns.

### Adjustable Precision

Change the digital brush size to create smaller detailed drops or larger dominant forms.

### Curated Colour Palette

Choose from a compact palette inspired by traditional and contemporary marbling compositions.

### 100-Phase Master Demonstration

Run an automated creative sequence consisting of:

- 40 pigment-placement phases
- 40 stylus-interaction phases
- 20 combing phases

The demonstration presents the project as a guided digital performance.

### English Voice Narration

The browser's Web Speech API provides spoken English narration during the master demonstration and cultural-information presentation.

Voice availability and pronunciation may differ depending on the browser and operating system.
